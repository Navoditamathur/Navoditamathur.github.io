* Pitt Digital Scholarship: (Jan 2023 – April 2023)
The project was to develop a web application to search for people, units, areas, methods, tools, and resources required for scholarships with special privileges given depending on the permission level, to add and delete the same.  My role was to query and construct the  graph database using Neo4j, API construction for CURD operations on the SQL database, and deployment of the website using Google Cloud.

* PPG Paint Colors: Final Project: (Jan 2023 – April 2023)
The project was  to identify important properties in real dataset provided by PPG as part of academic credit using R. It was an individual project and involved training about 25 models using resampling for both regression and classification.

* Optimal Path Routing using Reinforcement Learning: (Jan 2023 – April 2023)
The project aimed  to develop an environment with Open-AI Gym and create an AI (Artificial Intelligence) system to help control the flow of network traffic dynamically. I was primarily involved in displaying the traffic and routing paths including packets using PyQt.

* Recipe Search Engine: (Aug 2022 – Nov 2022)
The project’s goal was to design a search engine to search for recipes through the dataset using title and ingredients. The website was made using Django. Indexing was done using Python using  ntlk libraries for tokenization and removal of stop words.

* Crop Suitability Prediction: (Feb 2019-April 2019)
This project dealt with a detailed study aimed at developing a solution to bring farming to the doorsteps of everyday people with the help of machine learning. Primarily developed classification machine learning model which could predict the most suitable crop to grow. A web app was developed to input and store details like address, the products (seeds, fertilizers, pesticides, insecticides, tools) and their prices. The user/farmer is required to enter the details like region, temperature, rainfall, altitude, if the land was irrigated etc., and the webapp predicts the crop best suited to grow, given the conditions. It also provides a list of users providing the equipment required. Used MySQL database to store the registered users, Python for decision tree classification and android application to help guide further.

* App-based Empty Parking Spot Detection :(Jan 2019-Feb 2019)
The project was aimed to detect empty vehicle spots in parking by mobile based application with the help of sensors implemented using an Arduino board. Wrote the Arduino code to help detect cars using UV sensors and to change the color of LEDs.  The data was visualized on mobile app using Bluetooth.

* Password Cracker: (May 2023-July 2023)
Developed a website to crack and validate the password using MD5 of registered users and validate the password as strong, moderate, and weak. To do so a list of 2000 commonly used passwords was used and tested using permutations with 0-9 digits and special characters. To optimize, the password was restricted to use at most 2 numbers and 2 special characters.
